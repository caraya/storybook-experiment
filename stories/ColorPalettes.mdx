# Color Palettes

  ## Chroma Palettes

  Since we're working with OKLCH colors, we can create chromatic scales by just changing the chroma value (the second value in oklch colors).

  These palettes use the [color-mix](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/color-mix) function to create mixes of colors.
